# 🏏 Virat Kohli – The Test Maestro

A modern, responsive tribute website dedicated to **Virat Kohli**, celebrating his remarkable Test cricket career, leadership, achievements, iconic innings, and lasting legacy.

## 🌐 Live Demo

https://oibsip-virat-test-maestro.vercel.app/

---

## ✨ Features

- 🏏 Modern Hero Section
- 👤 About Virat Kohli
- 📅 Test Career Timeline
- 📊 Test Career Statistics
- 🏆 Historic Test Records
- 💯 Greatest Test Innings
- 👑 Captaincy Legacy
- 🥇 Awards & Honours
- 💬 Inspirational Quotes
- 📸 Responsive Image Gallery
- 📱 Fully Responsive Design
- ✨ Smooth Scroll Animations
- 🎨 Glassmorphism UI
- 🌙 Premium Dark Theme

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)
- Font Awesome
- Google Fonts (Poppins)
