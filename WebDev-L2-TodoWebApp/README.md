# 📝 Smart Todo Web App

A modern, responsive, and feature-rich **To-Do Web Application** built using **HTML, CSS, and Vanilla JavaScript**. The application helps users organize daily tasks with priority levels, due dates, search functionality, task filtering, and persistent local storage.

## 🌐 Live Demo

https://oibsip-smart-todo.vercel.app/

---

## ✨ Features

- ➕ Add New Tasks
- ✏️ Edit Existing Tasks
- 🗑️ Delete Tasks
- ✅ Mark Tasks as Completed
- ⭐ Priority Levels (High, Medium, Low)
- 📅 Due Date Selection
- 🔍 Search Tasks
- 📂 Filter Tasks (All / Pending / Completed)
- 📊 Task Statistics Dashboard
- 💾 Local Storage Support
- 📱 Fully Responsive Design
- 🎨 Modern Dark Theme
- ✨ Smooth Animations
- ⬆️ Back-to-Top Button
- 🔔 Toast Notifications

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)
- Local Storage API
- Font Awesome
- Google Fonts (Poppins)
