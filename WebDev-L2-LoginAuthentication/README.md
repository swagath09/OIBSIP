# 🔐 AuthFlow - Login Authentication System

A modern and responsive **Login Authentication System** built using **HTML, CSS, and JavaScript**. This project demonstrates a complete front-end authentication flow using **Local Storage** without any backend or database.

## 🌐 Live Demo

https://oibsip-authflow.vercel.app/

---

## ✨ Features

- 🔑 User Registration
- 🔐 Secure Login Authentication
- 👁️ Show / Hide Password
- 📧 Email Validation
- ✅ Confirm Password Validation
- 🔒 Password Strength Check
- 💾 Local Storage Authentication
- 👤 User Dashboard
- 🕒 Login Time Display
- 🚪 Logout Functionality
- 📱 Fully Responsive Design
- 🎨 Modern UI with Smooth Animations

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)
- Local Storage API
- Font Awesome
- Google Fonts (Poppins)
